# 사무실 온도 실시간 표시 웹

사무실의 온도를 IOT로 센서를 이용해 실시간으로 표시합니다.
데모 사이트에 표시되는 온도는 더미 데이터입니다.

## Git Clone

```git clone https://github.com/sungkuk5420/TemperatureIOT.git```

## NPM Install

```npm install```

## Start

```npm start```

## Build

```npm run-script build```
